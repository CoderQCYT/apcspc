# APCSPC (AP Computer Science Principles Compiler)
A real implementation of the psuedocode found on the [AP Computer Science Principles Exam](https://apcentral.collegeboard.org/media/pdf/ap-computer-science-principles-exam-reference-sheet.pdf), written in C++.

## Features
- Everything from the AP reference sheet
- Real Pratt parsing
- Cross platform by default
- UTF-8 Support
